- **Remote daemon** — better job control, real-time streaming, artifact management

- **Team mode** — shared machines, role-based access, audit logs
- **Fleet mode** — manage 10+ edge devices from one config
- **MCP server** — let AI agents discover and use your remote machines
- **VS Code / Cursor extension** — one-click remote development setup
